# 产品路线图

## 阶段 1：教学型 MVP

已完成：静态时间树、搜索、节点详情、类群聚焦、化石叠加、物种共同祖先比较、数据来源说明。

## 阶段 2：真实数据管线

- Catalogue of Life / GBIF 名称快照
- Open Tree of Life 子树 API
- TimeTree 年代查询与缓存
- PBDB 化石记录导入
- Wikimedia Commons 授权元数据
- 自动冲突报告和数据版本页面

## 阶段 3：百万节点渲染

- React + TypeScript
- Web Worker 解析和布局
- Canvas / WebGL 分层渲染
- 服务端按视口与类群裁剪子树
- 节点聚合、LOD 和增量加载
- URL 可分享的类群、节点与比较状态

## 阶段 4：学习与研究功能

- 地质年代尺和大陆漂移背景
- 关键性状演化映射
- 趋同演化专题
- 大灭绝前后多样性变化
- 物种分布地图
- 生态关系网络
- 古人类基因交流网络
- 系统发育争议的多方案切换
- 课程路线、测验和收藏笔记

## 阶段 5：开放协作

- 专家审核工作台
- 节点纠错与证据提交
- 数据来源差异对比
- 可导出的 Newick、Nexus、CSV 和 JSON
- 项目 API 与嵌入式小组件
