# EvoAtlas · 生物分类进化图谱

EvoAtlas 是一个纯前端交互式“生命之树”网站，用地质时间、系统发育分叉、代表性化石与最近共同祖先比较，直观展示不同生物类群的演化关系。

## 功能

- 地质时间系统发育树：拖拽、滚轮缩放、自动适应画布
- 对数/线性时间尺度切换
- 聚焦细菌、古菌、植物、真菌、动物、脊椎动物、哺乳动物、灵长类和人类谱系
- 按中文名、学名和分类等级搜索
- 区分现生类群、灭绝类群、共同祖先和干群节点
- 叠加五次大灭绝事件
- 节点详情：分类路径、年代、性状、置信度与来源
- 任意两个代表物种的最近共同祖先比较
- 深色/浅色主题和响应式布局
- 106 个教学型节点、16 个代表物种、5 个化石类群
- 无第三方前端依赖，可直接静态部署

## 本地运行

```bash
python3 -m http.server 8000
```

访问 `http://localhost:8000`。

## 数据结构

运行时数据位于 `data/data.js`。为减少静态文件体积，基础分类节点以结构化数组声明，重复的聚合支系由脚本生成，最终得到 106 个节点。每个节点包含：

```js
{
  id, parent, zh, latin, rank, ageMa,
  clade, status, confidence,
  summary, traits, sources
}
```

灭绝类群还可包含 `firstMa` 和 `lastMa`。当前数据用于教学和交互验证，不是完整物种名录。

## 数据来源规划

| 数据层 | 推荐来源 |
|---|---|
| 接受名、异名、分类等级 | Catalogue of Life、GBIF |
| 系统发育拓扑 | Open Tree of Life |
| 分化时间 | TimeTree |
| 基因组关联 | NCBI Taxonomy |
| 化石记录 | Paleobiology Database |
| 性状与生态 | Encyclopedia of Life |
| 图片与许可证 | Wikimedia Commons |

详见 `docs/data-sources.md`、`docs/methodology.md` 和 `docs/roadmap.md`。

## 数据校验

需要 Python 3 和 Node.js：

```bash
python3 scripts/validate_data.py
```

校验重复 ID、缺失父节点、循环引用、状态字段、时间范围和节点总量。

## 部署

仓库包含 `netlify.toml`，可直接连接 Netlify；发布目录为仓库根目录，无构建命令。

## 科学表达原则

1. 进化是分叉过程，不是从“低等”到“高等”的阶梯。
2. 现生物种通常共享共同祖先，而不是一个现生物种变成另一个。
3. 分化时间是估计值，应保留范围、来源和不确定性。
4. 多分叉可能表示关系尚未解析，不应为视觉整齐而任意二叉化。
5. 水平基因转移、杂交和基因渗入需要网状演化视图补充单棵树。
6. 化石类群默认表示近缘、旁支或干群，不自动宣称为确定直系祖先。
